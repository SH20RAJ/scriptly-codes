## 👾 Bring Nostalgic Charm to the Modern Web

Tired of generic, flat SaaS-style portfolios? PixelCraft is designed for developers, pixel artists, indie game creators, and retro aesthetics lovers who want their online home to make a bold visual statement.

PixelCraft combines 8-bit/16-bit retro console graphics with a blazing-fast, modern development stack. Behind the blocky borders, custom mouse cursor, and simulated CRT computer screen details lies a highly optimized Next.js App Router project utilizing TailwindCSS, static export (SSG), and responsive layout modules.

### Key Highlights

* **Authentic Retro Styling**: Built-in SVG computer terminal animations, responsive skill bars, blocky buttons, and localized pixel graphics.
* **Modern Performance**: Page transitions optimized in pure CSS. Replaced heavy JS animations to achieve near-zero route navigation times.
* **Full-featured Blog**: Preconfigured dynamic route templates to share your retro coding journey or design logs.
* **Centralized Data (lib/data.ts)**: Centralized records make it incredibly easy to swap out the portfolio showcase and blog articles in seconds.
* **Static HTML Export**: Preconfigured static site generation (SSG) outputs optimized files to the `out/` directory, ready to deploy instantly on GitHub Pages, Vercel, Netlify, or Surge.
* **Theme Support**: Seamless blocky transitions between dark and light modes.
* **Robust Custom Cursor**: Silky smooth GPU-accelerated cursor on desktop that automatically disables on touch screen mobile devices.
