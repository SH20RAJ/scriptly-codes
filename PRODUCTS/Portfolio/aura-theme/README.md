## 🎨 Customization Guide

### 1. Changing Text
Open `index.html`, `campaigns.html`, or `contact.html` in any text editor. Look for the text inside `<h2>`, `<h3>`, and `<p>` tags and replace it with your own content.

### 2. Changing Images
Navigate to the `assets/` folder. You will see images named `01.jpg` through `06.jpg`.
To use your own images, simply crop your images to a portrait ratio (e.g., 800x1200px), name them `01.jpg` to `06.jpg`, and overwrite the existing files in the `assets/` folder.

### 3. Changing Colors
Open `styles.css`. At the very top, you will see the `:root` variables.
Change `--accent: #B89676;` to your brand's primary color (e.g., a hex code for gold, neon green, or blue).
