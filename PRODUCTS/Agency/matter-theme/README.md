## 📦 Quick Start (1-2-3 Setup)

Since this is a lightweight, pure HTML/CSS/JS template, there is **zero build tool overhead** (no node_modules, no Webpack, no React setup needed!).

1. **Unzip** the download package.
2. **Open** `index.html` directly in your web browser to test it locally.
3. **Deploy** the folder to your hosting provider (Vercel, Netlify, GitHub Pages, or Surge) instantly!

## 🎨 Customization Guide

### 1. Changing Text & Copy
Open `index.html` in any text editor (like VS Code or Notepad) and search for the headings, descriptions, and testimonials to edit them directly.

### 2. Replacing Images & Videos
- All images are located in the `assets/` folder (or loaded from CDN).
- To replace an image, simply add your new image to the `assets/` folder and update its path in the `src="..."` attributes inside `index.html`.

### 3. Modifying Styles & Colors
Open `styles.css` to modify the visual aesthetics, margins, and typography sizes. The design uses standard CSS custom properties (variables) at the top of the file for quick global overrides.
