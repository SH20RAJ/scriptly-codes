# AURA — Premium AI SaaS Landing Page Template

Aura is a high-fidelity, futuristic dark-theme landing page template designed for modern AI productivity platforms, developer tools, and SaaS startups. Built with clean, responsive Tailwind CSS classes and vanilla JavaScript, it is lightweight, lightning-fast, and extremely simple to customize.

## Key Features
- **Interactive Particles Background**: Custom HTML5 canvas animation that updates smoothly on the fly.
- **Tailwind CSS Powered**: Styled using modular Tailwind CSS grids, flexboxes, and transition effects.
- **Copywriting Ready**: Fully professional copy for SaaS marketing layouts—no generic placeholders.
- **Infinite Marquee Integrations**: Scrolling brand banners using native keyframes.
- **Monthly/Yearly Pricing Switcher**: Dynamic slider toggle with client-side price updates.
- **Responsive Layout**: Designed to look stunning on mobile, tablet, and wide desktop screens.

## Getting Started

### 1. View Locally
Simply double-click the `index.html` file or open it in your browser of choice.

### 2. Live Web Deployment
To publish Aura online instantly, you can use Surge.sh:
1. Open your terminal inside this folder.
2. Install Surge:
   ```bash
   npm install -g surge
   ```
3. Run the deployment:
   ```bash
   surge . aura-ai-template.surge.sh
   ```

## Customize Colors
Aura uses customized brand configurations inside the inline config block. Edit `tailwind.config.theme.extend.colors` in `index.html` to swap brand colors (e.g. violet `#8b5cf6` with orange or emerald) instantly!
