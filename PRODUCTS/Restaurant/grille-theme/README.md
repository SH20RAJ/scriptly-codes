# GRILLÉ — Premium Fine Dining & Restaurant HTML Template

GRILLÉ is a premium, fully responsive luxury fine dining and restaurant HTML website template. Crafted with rich aesthetics, smooth animations, parallax interactive components, and premium typography, it offers a high-end interface for Michelin-grade restaurants, bistros, and culinary spaces.

## Features
- **Modern Serif Typography**: Using elegant, premium fonts suited for high-end dining.
- **Dynamic Hero Slider**: Beautiful sliding banners with entrance letter-reveal transitions.
- **Bespoke Animations & Parallax**: Parallax hovering visual elements, smooth scroll-triggered navigations, and hover-state zoom scales.
- **Interactive Forms**: Built-in structured reservation and newsletter forms.
- **Perfect Responsiveness**: Tailored layout grids adapting flawlessly to mobile, tablet, and desktop viewports.

## File Structure
- `index.html` — Clean semantic structure with premium copy.
- `assets/css/style.css` — High-performance responsive layouts and styling variables.
- `assets/js/script.js` — Parallax triggers, slider functionality, mobile navigation toggling, and lazy loads.
- `assets/images/` — Structured image folders and SVG assets.

## Getting Started
To view the template locally, you can serve it using any local development server (e.g., `Live Server` in VSCode or running `npx serve .` in the terminal).
